# Installation

`npm i shadowwizardmatu --save`

Then...

```js
import { shadowwizard } from 'shadowwizardmatu'

shadowwizard({
    shadow_type: 'soft',
    padding: false,
})
```

## Options

-   _shadow_type_ - _hard | soft_
-   _padding_ - _boolean_
